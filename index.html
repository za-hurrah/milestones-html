Getting started

To get started with coding, you'll need a terminal for your device to carry out commands, and a software to write code on. On Windows, you can download GitBash to access terminal, and Visual Studio to write your code in

GitBash

Visual Studio Code

The terminal allows you to create commands for you device to follow. Open GitBash to access the command line.
The first thing you'll want to do is locate your commands in a file folder. To reach a specific file, or directory, type cd and then the name of the directory. Here is an example:

There are a variety of commands to use to navigate directories. In the current location, typing in

  • "ls" will list out directories
  • "mkdr ___" with a directory name will make a new folder
  • "fsutil file createnew __ 0" with a file name will make a new empty file (This is for Windows)
  • "code ." will open up the code for the file in VS Code
